#### 安装教程(windows)

1. 克隆repo
2. 安装 [pixi](https://github.com/prefix-dev/pixi/releases/latest/download/pixi-x86_64-pc-windows-msvc.msi)
#### 使用说明

```bash
cd LEL
#安装依赖
pixi shell
#设置你的api keys
copy .env_template .env
#在.env中填入你的api keys
```
运行`main.ipynb`即可
